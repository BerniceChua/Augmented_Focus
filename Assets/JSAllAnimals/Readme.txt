Thank you for downloading JS all animals pack.

This package contains 3d model and animations of animals.
This package contains
5 invertebrate(Ammonite, Anomalocaris,Ceratocanthus, Shrimp, Snowcrab)
2 amphibians(FrogAndSkeleton, PoisonDartFrog)
14 birds(Archaeopteryx, BaldEagle, BarnOwl, BowerBird, Cassowary, HarpyEagle, HummingBird, Penguin, Pigeon, RedCrownedCrane, Swan, Ostrich, Secretarybird, Moa)
7 fishes(Dunkleosteus, Sardine, Seahorse, Angelfish, Arowana, Pirarucu, Shark)
14 mammals(Deer, Capybara, Maned wolf, Orca, Bear, Gorilla, Jerboa, Earless seal, Rhinoceros, Giraffe, Lion, Wildebeest, Elephant, Paraceratherium).
14 reptiles(2 dinosaurus(AllosaurusAndSkeleton, Gallimimus), Mosasaurus, 4 lizards(Gecko, Iguana, Komododragon, Leaf-tailed Gecko),6 turtles
(AlligatorSnappingTurtle, BoxTurtle, RedfootedTurtoise, SeaTurtle, AfricanSpurredTortoise, Giant tortoise), Crocodile).
