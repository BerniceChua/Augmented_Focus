Thank you for downloading.

-Modals folder contains BarnOwl.fbx and BarnOwlNoFurModel.fbx.
BarnOwl.fbx contains model and animations of barn owl.
BarnOwlNoFurModel.fbx contains model of barn owl that does not have face and leg feather.
-Walk animations are compatible with root motion.
-Scenes folder contains a demo scene.
-Prefabs folder contains a prefab that was used in the demo scene.
